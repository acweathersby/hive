<h1 align=center> HIVE </h1>

## Sandbox Engine

Hive is a general purpose sandbox for creating digital content with a focus on 2D based images.

Written in C++

Capabilities:

- Node and Graph Tools
- Audio
- Text
- Input
- File Saving
- Custom Scripting, possibly based on Lua
- OpenGL graphics system.
- Note System?
